# Microbio

📌 **Sobre o projeto**

Este repositório contém o desenvolvimento de um projeto acadêmico da disciplina de **Projeto Integrador**.

O objetivo do projeto é criar uma solução para a empresa **Microbio**, focada na análise de laudos a partir de orçamentos enviados por seus clientes.

---

## 🎯 Objetivo

A proposta é desenvolver um sistema que permita:

- Receber orçamentos enviados pelos clientes
- Realizar a análise dessas informações
- Auxiliar na organização de laudos solicitados

---

## 🛠️ Tecnologias utilizadas

- **Java**
- **Spring Boot**
- **Thymeleaf**
- **PostgreSQL**
- **Git** (Git Flow)
